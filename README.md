## ⚖ 玩法機制

- **心神 ❤ × 5**：答錯 −1 心神並 **−10 藍圖能量**；耗盡即壞結局
- **提示 🔍 × 3**：每題可求一次教學式提示
- **藍圖能量 ⚡**：答對 +20；決定結局位階（規矩天成・一代宗師／匠心獨運／初窺門徑）
- **🏺 尋寶收集**：六街區各藏一枚金色光點，收集六件上古數學文物（青銅圓規、魯班墨斗、青銅矩尺、司南、老算盤、九章殘簡）入「寶物圖鑑」
- 進度與圖鑑自動存檔（localStorage）
